# Content Difference Checker

A free online tool to compare text, files, URLs, JSON, HTML and documents directly in your browser.

## Features

* Compare text instantly
* File comparison support
* URL comparison
* Side-by-side diff view
* Highlight added and removed content
* No signup required
* Browser-based processing
